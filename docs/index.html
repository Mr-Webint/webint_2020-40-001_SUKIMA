九州の学生と企業がつながるプラットフォーム

SUKIMAは提供いたします。
企業が可能性豊かな学生とつながる場。

SUKIMAは、九州の学生キャリア支援事業を行なっております。主に新卒採用を実施している企業様と学生が出会う採用イベントを企画 / 運営しており、大小様々な規模のイベントで優秀な学生たちと繋がるお手伝いをいたします。

私たちのサービスが秘める可能性

SUKIMAが提供する人材獲得の"場"をご紹介いたします。企業さまのご要望に応じた小規模・個社イベントの「SUKIMA BOX」と大規模マッチングイベント「SUKIMA DOME」でこれからを担う学生と出会いましょう。

SUKIMA DOME

学生40名以上と1回のイベントで出会うことが可能です。学生にとって有益なテーマのイベントを開催することで、イベント参加企業様に対する学生の満足度も高くなっています。

SUKIMA BOX

学生10〜15名と密にコミュニケーションが取れるイベントです。カフェやバーを利用して企業様毎にカスタマイズされたコンテンツを提供するため、企業様の魅力をカジュアルに学生に伝えられます

規模だけでなく
テイストも多彩なサービスを提供。

- SUKIMA BAR -

毎週日曜夜に学生同士がキャリアをテーマに交流する場です。毎週10名程度の学生が集まります。企業様が参加してコンテンツを企画することも可能ですので、リアルな学生コミュニティにアクセスしたい企業様は是非ご利用下さい。

- cenco -

若者とクリエイターの共創空間。若者の「やってみたい」を実行する場、プロジェクトを立ち上げていく場としてのスペースと、クリエイターが集中してものづくりができるスペースが同じ空間に共存することで、さまざまなコラボを創り、いろんなプロジェクトが生み出されることをコンセプトとします。

- VENTURE EXPO -

ベンチャー万博in九州とは、九州の大学に在籍している就活生が国内のベンチャー企業の“働く”を少ない機会の中で多く知ってもらうことを目的としたイベントです。九州の就活生と厳選されたベンチャー企業のマッチングの機会を創出します。
